# 🛒 Ecommerce Sales Analysis (SQL)

## 🎯 Objective
Analyze ecommerce transaction data to evaluate revenue performance, customer behavior, and key business KPIs.

---

## 🛠 Tools Used
- SQLite
- DB Browser for SQLite
- SQL (JOIN, GROUP BY, Aggregations, Subqueries)

---

## 📊 Analysis Performed

### 💰 Revenue Analysis
- Total Revenue calculation
- Monthly Revenue trend analysis

### 👥 Customer Analysis
- Top spending customers
- Repeat vs One-Time customer analysis

### 📦 Order Metrics
- Average Order Value (AOV)

---

## 🔗 Database Structure
Multi-table relational model:

customers → orders → order_items → products  
orders → payments

---

## 📈 Key Insights
- Total revenue exceeded 16M
- Majority of customers were one-time buyers
- Average Order Value was approximately 160.99
- Revenue varied significantly across months
